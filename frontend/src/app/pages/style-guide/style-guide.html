Componente de tarjeta reutilizable con soporte para imagen, título, descripción y contenido personalizado. Disponible en variantes vertical (por defecto) y horizontal.

Cards Verticales

Layout vertical con imagen en la parte superior, ideal para mostrar cursos, artículos o productos.

Componente de botón con 4 variantes visuales y 3 tamaños. Incluye estados hover, focus, active y disabled.

Color amarillo/dorado. Para acciones principales como "Guardar", "Enviar", "Continuar".
